# Javali Replica Project

A minimal React frontend for a Drupal content site.

This repo contains a simple React app built with Vite and Tailwind CSS that fetches content from Drupal JSON:API.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Then open the local URL shown by Vite in your browser.

## What is included

- `frontend/src/App.jsx` — main React app
- `frontend/src/index.css` — Tailwind import and basic styling
- `frontend/src/main.jsx` — React entry point
- `frontend/postcss.config.js` — PostCSS setup for Tailwind
- `frontend/package.json` — dependencies and scripts
